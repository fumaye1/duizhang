# 多云仓自动对账工具（MVP）分发说明

本说明用于把项目分发给同事电脑进行演示，面向 Windows 10/11 + PowerShell。

## 一、分发方式（推荐顺序）

### 方式 A：直接分发项目文件夹（推荐）
适合内网演示、快速试用。对方电脑安装 Python 后即可运行。

### 方式 B：局域网演示（你电脑做服务端）
你本机启动服务，其他电脑浏览器访问你的 IP。

### 方式 C：制作“一键启动包”（可选）
后续可加启动脚本或简易安装包，但 Streamlit 打包成 exe 往往不稳定，建议先用方式 A/B。

---

## 二、方式 A：分发项目文件夹

### 1) 打包给对方
把整个项目文件夹打包给对方（必须包含以下文件）：
- `app.py`
- `reconcile.py`
- `requirements.txt`
- `README.md`

### 2) 对方电脑安装 Python
在对方电脑参考 `SETUP.md` 安装 Python 3.11 并配置环境。

### 3) 对方电脑安装依赖并启动
在 PowerShell 执行：

```powershell
cd "<项目解压路径>\duizhang"
python -m pip install -r requirements.txt
python -m streamlit run app.py --browser.gatherUsageStats false
```

浏览器访问：`http://localhost:8501`

---

## 三、方式 B：局域网演示（你电脑做服务端）

### 1) 你电脑启动服务
在你的电脑执行：

```powershell
cd "<你的项目路径>\duizhang"
python -m streamlit run app.py --server.address 0.0.0.0 --server.port 8501 --browser.gatherUsageStats false
```

### 2) 查询你的 IP
PowerShell 执行：

```powershell
ipconfig
```

找到你电脑的 IPv4 地址，例如 `192.168.1.23`。

### 3) 其他电脑访问
在其他电脑浏览器输入：

```
http://<你的IP>:8501
```

示例：`http://192.168.1.23:8501`

## 四、常见问题

### 1) 浏览器提示“localhost 拒绝连接”
- 可能是首次启动卡在 Streamlit 的 `Email:` 交互输入。
- 处理：在终端直接回车，或使用 `--browser.gatherUsageStats false` 参数启动。

### 2) 端口无法访问
- 请确认防火墙允许 `8501` 端口。
- Windows 首次弹窗时点击“允许访问”。

### 3) python 命令不可用
- 请按 `SETUP.md` 执行一键脚本并重启终端。

# 多云仓自动对账工具（MVP）

基于 Streamlit 的主对账功能 MVP，固定字段名，快速验证业务流程。

## 运行
```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 固定字段名（必需）

### 旺店通发货明细
- 物流单号
- 商家编码
- 数量
- 收货省份
- 发货时间
- 快递公司
- 实际重量（可选）
- 店铺名称（可选）

### 云宝发货明细
- 物流单号
- 货品名称
- 数量
- 收货地址
- 发货时间
- 快递公司
- 实际重量（可选）
- 店铺名称（可选）

### 云宝名称货品表
- 货品名称
- 商家编码

### 毛重表
- 商家编码
- 毛重(g)
- 箱规

### 云仓重量段定义表
- 云仓
- 重量段结束(kg)

### 多条件资费表
- 云仓
- 快递公司
- 重量段结束(kg)
- 是否打包品
- 省份
- 生效开始日期
- 生效结束日期
- 快递费(元)（或快递费）

### 云仓账单
- 物流单号
- 计费重量(kg)
- 快递费(元)
- 包装费(元)（可选）
- 云仓

### 可选表
- 耗材表：商家编码、价格(元)
- 撕单表：物流单号
- 售后赔付表：物流单号、赔付金额(元)

## 输出
- 对账结果：明细对账与核算运费
- 汇总：按云仓+快递公司汇总
- 异常清单：资费未匹配、重量段缺失、账单缺失

## 字段映射模板

### 使用方式
- 在页面顶部“字段映射模板”区域上传/下载 JSON 模板。
- 模板会自动回填到各数据源映射下拉框。
